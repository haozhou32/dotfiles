# Tmux initiatation

 To initiate tmux, follow these steps:

*Step 0*. Add the following code to ~/.zshrc (This step is required only on macOS):
```shell
export XDG_CONFIG_HOME="$HOME/.config"
```

*Step 1*. Install tmp plugin manager:
 Run the following code:
```shell
 git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm
```

*Step 2*. To install plugins:
 Start a tmux session and press prefix + I
